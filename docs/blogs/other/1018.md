```sql
主键约束 	PRIMARY KEY
外键约束 	FOREIGN KEY
默认值约束 	DEFAULT 
自增ID 	IDENTITY(1,1) 第一个种子值，第二个步长

```

### 例子

```sql
CREATE TABLE  zuo
(ID  INT IDENTITY(1,1) PRIMARY KEY,
COL  INT DEFAULT(100),
SEX  INT CHECK(SEX=0 OR SEX=1),
RAD INT FOREIGN KEY REFERENCES S_id(id))
```
